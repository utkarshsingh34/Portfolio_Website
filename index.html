Current Projects

AI Evidence Auditor

Python, TensorFlow, Tesseract, OpenCV, Hugging Face, Azure

First time working with an industry dev team. The goal of this project is to develop a CPU-optimized, cost efficient end-to-end AI solution to automate manual auditing of cloud configuration compliance.

  • Learning through building. Training a CNN model on a loaded dataset using keras to classify images as relatable or unrelatable for an initial validation check.
  • Images classified as relatable are preprocessed by converting to grayscale and applying thresholding to enhance text visibility. Performing Optical Character Recognition (OCR) on pre-processed images using Tesseract for text extraction.
  • Researching and learning to apply Natural Language Processing via DistilledBERT from the Hugging Face library to automatically audit extracted text from the previous step.
  • Utilizing Redis to cache calls to Azure and AWS policy configuration APIs for real-time compliance checks.
